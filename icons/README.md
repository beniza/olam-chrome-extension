# Olam Dictionary Chrome Extension Icons

This directory should contain the extension icons in the following sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Creating Icons

You can create icons using:
1. **Online tools**: 
   - [Favicon Generator](https://favicon.io/)
   - [Canva](https://www.canva.com/)
   
2. **Design software**:
   - Adobe Illustrator
   - Figma
   - GIMP (free)

## Icon Design Suggestions

Based on olam.in branding:
- Use a blue color scheme (#4299e1 or #2c5282)
- Consider incorporating Malayalam script or a book/dictionary symbol
- Keep it simple and recognizable at small sizes
- Ensure good contrast for visibility

## Temporary Solution

Until you create custom icons, you can:
1. Use a simple colored square as placeholder
2. Download the olam.in favicon and resize it
3. Create icons using the online tools mentioned above

Simply place your icon files in this directory with the exact names specified above.
